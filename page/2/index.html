numactl

发表于 2017-12-02 |

介绍

NUMA(Non-Uniform Memory Access)字面直译为“非一致性内存访问”
对于多核CPU而言，它的作用可以将程序绑定在固定的CPU的进程以及固定的内存块上，避免了程序在不同CPU和内存块之间切换导致的时间消耗。
从而可以提高程序的性能
